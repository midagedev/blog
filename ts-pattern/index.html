ts-pattern

ts-pattern 라이브러리는 TypeScript를 위한 전체적인 패턴 매칭 라이브러리입니다. 이 라이브러리는 패턴 매칭의 강력함과 타입 안정성을 결합하여 TypeScript에서 패턴 매칭을 쉽게 사용할 수 있게 해줍니다.

아래는 ts-pattern를 사용하는 몇 가지 예제입니다. 사용하기 전에 라이브러리를 설치해야 합니다.


npm install ts-pattern --save
  1. 기본적인 패턴 매칭

import { match } from 'ts-pattern';

const x = 10;

const result = match(x)
  .with(10, () => 'ten')
  .with(20, () => 'twenty')
  .otherwise(() => 'unknown');

console.log(result); // 'ten'
  1. 객체 패턴 매칭

type User = { type: 'user'; name: string; age: number };

const alice: User = { type: 'user', name: 'Alice', age: 25 };

const greeting = match(alice)
  .with({ type: 'user', name: 'Alice' }, (user) => `Hello, ${user.name}!`)
  .otherwise(() => 'Who are you?');

console.log(greeting); // 'Hello, Alice!'
  1. 배열 패턴 매칭

const numbers = [1, 2, 3];

const result = match(numbers)
  .with([1, 2, 3], () => 'One, Two, Three')
  .otherwise(() => 'Unknown sequence');

console.log(result); // 'One, Two, Three'
  1. 유니언 타입과 패턴 매칭

type Shape = { type: 'circle', radius: number } | { type: 'square', side: number };

const shape: Shape = { type: 'circle', radius: 5 };

const area = match(shape)
  .with({ type: 'circle' }, (s) => Math.PI * s.radius * s.radius)
  .with({ type: 'square' }, (s) => s.side * s.side)
  .otherwise(() => 0);

console.log(area); // 78.53981633974483
  1. 와일드카드 사용하기

const name = 'Bob';

const greeting = match(name)
  .with('Alice', () => 'Hello, Alice!')
  .with(_, () => 'Hello, Stranger!')  // _는 어떤 값이든 매치됩니다.
  .run();

console.log(greeting); // 'Hello, Stranger!'
  1. 옵셔널 값 다루기

type MaybeNumber = number | null | undefined;

const value: MaybeNumber = null;

const description = match<MaybeNumber, string>(value)
  .with(null, () => 'It is null')
  .with(undefined, () => 'It is undefined')
  .otherwise(v => `Value is ${v}`);

console.log(description); // 'It is null'
  1. 타입 가드 활용하기

function isCircle(shape: any): shape is { type: 'circle', radius: number } {
  return shape.type === 'circle';
}

const shape: Shape = { type: 'circle', radius: 5 };

const description = match(shape)
  .with(isCircle, circle => `Circle with radius ${circle.radius}`)
  .otherwise(() => 'Not a circle');

console.log(description); // 'Circle with radius 5'
  1. 날짜 패턴 매칭

const today = new Date();

const dayOfWeek = match(today.getDay())
  .with(0, () => 'Sunday')
  .with(1, () => 'Monday')
  .with(2, () => 'Tuesday')
  .otherwise(() => 'Some other day');

console.log(dayOfWeek);

ts-pattern을 사용하면 여러 유형의 데이터와 패턴을 효과적으로 매치하고 관련된 로직을 적용할 수 있습니다.
