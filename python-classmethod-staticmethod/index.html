python 클래스의 classmethod와 staticmethod

이 데코레이터들은 메서드의 동작 방식을 변경하여, 인스턴스 메서드가 아닌 클래스 레벨에서 동작하는 메서드를 정의할 수 있게 해줍니다.

  1. @classmethod :
  • 첫 번째 인자로 클래스 자체(cls)를 받습니다. 이를 통해 클래스 변수에 접근하거나 클래스 메서드를 호출할 수 있습니다.
  • 인스턴스를 생성하지 않고도 클래스 레벨에서 메서드를 호출할 수 있습니다.
  • Django 모델에서는, 특정 조건에 따른 쿼리셋을 반환하는 메서드나, 팩토리 메서드 등을 정의할 때 유용하게 사용될 수 있습니다.

from django.db import models

class Book(models.Model):
    title = models.CharField(max_length=100)
    published_date = models.DateField()

    @classmethod
    def get_recent_books(cls, days=30):
        recent_date = timezone.now() - timedelta(days=days)
        return cls.objects.filter(published_date__gte=recent_date)
  1. @staticmethod :
  • 클래스나 인스턴스를 참조하지 않는 메서드를 정의할 때 사용됩니다.
  • 이 메서드는 클래스의 상태나 인스턴스의 상태를 변경하지 않는, 순수한 함수처럼 동작합니다.
  • Django 모델에서는, 모델과 직접적인 관련이 없는 보조 함수나 유틸리티 함수를 정의할 때 사용될 수 있습니다.

class Book(models.Model):
    title = models.CharField(max_length=100)
    published_date = models.DateField()

    @staticmethod
    def is_valid_title(title):
        return len(title) > 0 and len(title) <= 100
