1. 서론 (Introduction) [Page 1]

함수형 프로그래밍은 함수의 적용을 기본 연산으로 사용하며, 주요 프로그램도 입력을 인수로 받고 출력을 결과로 제공하는 함수로 작성됩니다. 이 접근 방식은 소프트웨어의 복잡성을 줄이고 모듈화를 촉진하며, 디버깅을 쉽게 만듭니다. 또한, 함수형 프로그래밍은 일반적인 수학 함수와 유사하게 작동하므로, 수학적 추론이 더 쉽습니다.

예제 코드:


// 입력을 받아 출력을 반환하는 함수
const greet = (name) => `Hello, ${name}!`;
console.log(greet("John")); // 출력: Hello, John!


// 함수를 인수로 받아 다른 함수를 반환하는 고차 함수
const multiplyBy = (factor) => (number) => number * factor;
const multiplyByTwo = multiplyBy(2);
console.log(multiplyByTwo(3)); // 출력: 6

2. 구조화 프로그래밍과의 유사성 (An Analogy with Structured Programming) [Page 3]

함수형 프로그래밍은 구조화 프로그래밍과 유사한 이점을 제공합니다. 모듈화된 설계는 코드 재사용을 촉진하고 디버깅을 쉽게 만듭니다. 함수형 언어는 문제를 서브 문제로 나누고 해결책을 결합하는 새로운 방법을 제공하며, 이를 통해 코드의 모듈화를 향상시킵니다.

예제 코드:


// 모듈화된 함수
const add = (a, b) => a + b;
const multiply = (a, b) => a * b;
const operate = (operator, a, b) => operator(a, b);
console.log(operate(add, 2, 3)); // 출력: 5
console.log(operate(multiply, 2, 3)); // 출력: 6


// 재사용 가능한 모듈
const square = (x) => x * x;
const sumOfSquares = (a, b) => add(square(a), square(b));
console.log(sumOfSquares(3, 4)); // 출력: 25

3. 함수 결합 (Gluing Functions Together) [Page 4-8]

함수형 프로그래밍에서는 작은 함수를 복잡한 함수로 결합할 수 있습니다. 이를 통해 코드의 재사용성과 유지 보수성이 향상됩니다. 예를 들어, 리스트 처리와 관련된 다양한 작업을 수행하는 고차 함수를 작성할 수 있습니다.

예제 코드:


// 리스트의 합계 계산
const sum = (list) => list.reduce((acc, val) => acc + val, 0);
console.log(sum([1, 2, 3])); // 출력: 6


// 리스트의 모든 요소에 함수 적용
const map = (fn, list) => list.map(fn);
const double = (x) => x * 2;
console.log(map(double, [1, 2, 3])); // 출력: [2, 4, 6]

4. 프로그램 결합 (Gluing Programs Together) [Page 9]

함수형 프로그래밍에서는 "게으른 평가"를 사용하여 전체 프로그램을 결합할 수 있습니다. 이를 통해 프로그램 간의 엄격한 동기화가 가능하며, 필요한 만큼만 계산을 수행합니다. 게으른 평가는 강력한 모듈화 도구로 작용하며, 종료 조건과 루프 본문을 분리하는 데 유용합니다.

예제 코드:


// 게으른 평가를 사용한 무한 시퀀스
const infiniteSequence = function* () {
  let i = 0;
  while (true) yield i++;
};
const take = function (n, iter) {
  const result = [];
  for (const val of iter) {
    if (result.length === n) break;
    result.push(val);
  }
  return result;
};
console.log(take(5, infiniteSequence())); // 출력: [0, 1, 2, 3, 4]


// 두 프로그램의 결합
const double = (x) => x * 2;
const sum = (list) => list.reduce((acc, val) => acc + val, 0);
const doubleAndSum = (list) => sum(list.map(double));
console.log(doubleAndSum([1, 2, 3])); // 출력: 12
